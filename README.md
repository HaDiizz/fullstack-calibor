# Calibor Web Application

> [Calibor Demo](https://calibor.netlify.app/ "Calibor Demo")

## Project Usage

Clone Project
```shell
git clone https://github.com/HaDiizz/fullstack-calibor.git
```
Install Client
```shell
cd client
```
```shell
npm install
```
Install Server
```shell
cd server
```
```shell
npm install
```
> *Before run project, You need to create and config .env in both folders.*

Run Client Project
```shell
npm run dev
```
Run Server Project
```shell
npm start
```
